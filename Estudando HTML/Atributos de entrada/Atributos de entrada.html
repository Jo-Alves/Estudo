O atributo maxlength especifica o comprimento máximo permitido para o campo de entrada:

Com um atributo maxlength, o campo de entrada não aceita mais do que o número permitido de caracteres.

O atributo maxlength não fornece nenhum feedback. Se você deseja alertar o usuário, você deve escrever o código JavaScript.



O atributo de autocomplete especifica se um formulário ou campo de entrada deve ter ativado ou desativado o autocompletar. Quando o preenchimento automático está ativado, o navegador completa automaticamente os valores de entrada com base nos valores que o usuário já inseriu antes. Dica: é possível ter "preenchido" automaticamente para o formulário e "desligado" para campos de entrada específicos, ou vice-versa. O atributo de preenchimento automático funciona com ‹form› e os seguintes tipos de : texto, pesquisa, url, tel, email, senha, datepickers, alcance e cor.




O atributo de autofocus especifica que o campo de entrada deve obter automaticamente o foco quando a página é carregada.



O atributo list refere-se a um elemento ‹datalist› que contém opções pré-definidas para um elemento ‹input›.

O atributo requerid especifica que um campo de entrada deve ser preenchido antes de enviar o formulário.

O atributo required funciona com os seguintes ipos de entrada: texto, pesquisa, url, tel, email, senha, selecionadores de data, número, caixa de seleção, rádio e arquivo.



O atributo step especifica os intervalos de número legal para um elemento .

Exemplo: se step = "3", números legais podem ser -3, 0, 3, 6, etc.

Dica: O atributo passo pode ser usado junto com os atributos máximo e min para criar uma gama de valores legais.

atributo passo funciona com os seguintes tipos de entrada: número, intervalo, data, data-hora local, mês, hora e semana.
