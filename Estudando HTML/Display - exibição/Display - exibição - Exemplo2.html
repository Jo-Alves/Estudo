Este Painel contém um elemento div, oculto por padrão(exibição; nenhum). É desenhado com CSS e usamos JavaScript para mostrar e remover (display: block e display:none). Como funciona: observe que o elemento p com classe = "flip" tem um atributo onclick anexado a ele. Quando o usuário clicar no elemento p, é executada uma função chamada myFunction (), que altera o estilo do div com id = "panel" do display: none (hidden) para display: block (visible). Você aprenderá mais sobre JavaScript no nosso Tutorial de JavaScript.
